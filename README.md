# Split
## By Alex Coady

### Set up environment

- Install [mongodb](http://docs.mongodb.org/manual/tutorial/install-mongodb-on-os-x/)
- `mongod` in seperate terminal window
- Install [node](http://nodejs.org/)

### Set up webapp

- Clone this repo
- `cd to/repo`
- `npm install && bower install`
- `gulp develop`
- Go to http://localhost:3000
